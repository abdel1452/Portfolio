Vue d'ensemble

Nom de l'application : MauiAppCoach

Type : Application MAUI (.NET MAUI)

Objectif : Calcul de l'IMG (Indice de Masse Grasse)

Structure du projet

  • MauiAppCoach - Projet principal
  • MauiAppCoach.Core - Bibliothèque contenant le modèle Profil

Pages de l'application

1. MainPage

Fichiers : MainPage.xaml, MainPage.xaml.cs

Interface utilisateur (MainPage.xaml)

  • Layout : VerticalStackLayout centré avec espacement de 25 et marge de 10

Contrôles

Contrôle Description Propriétés
entPoids Entry pour saisir le poids en kg Clavier numérique, Police 22, DarkBlue
entTaille Entry pour saisir la taille en cm Clavier numérique, Police 22, DarkBlue
entAge Entry pour saisir l'âge Clavier numérique, Police 22, DarkBlue
rdHomme RadioButton "Homme" Sélectionné par défaut, Groupe "Sexe"
rdFemme RadioButton "Femme" Groupe "Sexe"
Button "Calculer IMG" Police 22, Événement btCalculer_Clicked
imgResultat Image pour afficher le résultat Hauteur 185, AspectFit
CounterLabel Label pour afficher l'IMG Police 18, Center, Black

Fonctionnalité (MainPage.xaml.cs)

Méthode btCalculer_Clicked :

  1. Convertit les valeurs saisies en entiers (poids, taille, âge)
  2. Détermine le sexe : 1 si Homme, 0 si Femme
  3. Crée une instance de Profil avec ces paramètres
  4. Affiche l'IMG dans CounterLabel avec format "0.0"
  5. Affiche l'image correspondante dans imgResultat
Note : Pas de gestion d'erreur pour les conversions

3. App

Fichiers : App.xaml, App.xaml.cs, MauiProgram.cs

  • App.xaml : Charge les ressources (Colors.xaml, Styles.xaml)
  • App.xaml.cs : Crée une fenêtre avec AppShell comme contenu
  • MauiProgram.cs : Configure l'application MAUI, enregistre les polices (OpenSans-Regular, OpenSans-Semibold), active le logging en mode DEBUG

Modèle métier : Profil

Fichier : MauiAppCoach.Core/Modele/Profil.cs

Propriétés privées

  • sexe : int
  • poids : int
  • taille : int
  • age : int
  • img : double
  • message : string
  • image : string

Constructeur

Profil(int sexe, int poids, int taille, int age) - Initialise les propriétés - Appelle CalculIMG() - Appelle ResultatIMG()
Note : Constructeur par défaut commenté dans le code

Méthodes privées

CalculIMG()

Calcule l'IMG avec la formule :

img = (1.2 * poids / (tailleEnMetres²)) + (0.23 * age) - (10.83 * sexe) - 5.4 où tailleEnMetres = taille / 100.0
ResultatIMG()

Détermine le message et l'image selon l'IMG :

Sexe Condition IMG Message Image
Femme (0) IMG < 25 "Trop maigre." smiley_tropmaigre.png
25 ≤ IMG < 30 "Parfait." smiley_parfait.png
IMG ≥ 30 "Surpoids." smiley_surpoids.png
Homme (1) IMG < 15 "Trop maigre." smiley_tropmaigre.png
15 ≤ IMG < 20 "Parfait." smiley_parfait.png
IMG ≥ 20 "Surpoids." smiley_surpoids.png
