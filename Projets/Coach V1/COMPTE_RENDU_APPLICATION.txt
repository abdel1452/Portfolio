--------------------------------------------------------------------------------

1. MauiAppCoach - Projet principal

PAGES DE L'APPLICATION
--------------------------------------------------------------------------------

1. MAINPAGE
--------------------------------------------------------------------------------

Fichiers : MainPage.xaml, MainPage.xaml.cs

Interface utilisateur (MainPage.xaml) :
- Layout : VerticalStackLayout centré avec espacement de 25 et marge de 10

Contrôles :
- entPoids : Entry pour saisir le poids en kg
  * Clavier numérique
  * Taille de police : 22
  * Couleur du texte : DarkBlue

- entTaille : Entry pour saisir la taille en cm
  * Clavier numérique
  * Taille de police : 22
  * Couleur du texte : DarkBlue

- entAge : Entry pour saisir l'âge
  * Clavier numérique
  * Taille de police : 22
  * Couleur du texte : DarkBlue

- rdHomme : RadioButton "Homme" (sélectionné par défaut)
- rdFemme : RadioButton "Femme"
  * Groupe : "Sexe"

- Button "Calculer IMG"
  * Taille de police : 22
  * Événement : btCalculer_Clicked

- imgResultat : Image pour afficher le résultat
  * Hauteur : 185
  * Aspect : AspectFit

- CounterLabel : Label pour afficher l'IMG
  * Texte initial : "Votre IMG :"
  * Taille de police : 18
  * Alignement : Center
  * Couleur : Black

Fonctionnalité (MainPage.xaml.cs) :

Méthode btCalculer_Clicked :
1. Convertit les valeurs saisies en entiers (poids, taille, âge)
2. Détermine le sexe : 1 si Homme, 0 si Femme
3. Crée une instance de Profil avec ces paramètres
4. Affiche l'IMG dans CounterLabel avec format "0.0"
5. Affiche l'image correspondante dans imgResultat

Note : Pas de gestion d'erreur pour les conversions

MODÈLE MÉTIER : PROFIL
--------------------------------------------------------------------------------

Fichier : MauiAppCoach.Core/Modele/Profil.cs

Propriétés privées :
- sexe : int
- poids : int
- taille : int
- age : int
- img : double
- message : string
- image : string

Constructeur :
Profil(int sexe, int poids, int taille, int age)
- Initialise les propriétés
- Appelle CalculIMG()
- Appelle ResultatIMG()

Note : Constructeur par défaut commenté dans le code

Méthodes privées :

CalculIMG() :
Calcule l'IMG avec la formule :
img = (1.2 * poids / (tailleEnMetres²)) + (0.23 * age) - (10.83 * sexe) - 5.4
où tailleEnMetres = taille / 100.0

ResultatIMG() :
Détermine le message et l'image selon l'IMG :

Pour les femmes (sexe = 0) :
- IMG < 25 : message = "Trop maigre.", image = "smiley_tropmaigre.png"
- 25 ≤ IMG < 30 : message = "Parfait.", image = "smiley_parfait.png"
- IMG ≥ 30 : message = "Surpoids.", image = "smiley_surpoids.png"

Pour les hommes (sexe = 1) :
- IMG < 15 : message = "Trop maigre.", image = "smiley_tropmaigre.png"
- 15 ≤ IMG < 20 : message = "Parfait.", image = "smiley_parfait.png"
- IMG ≥ 20 : message = "Surpoids.", image = "smiley_surpoids.png"

Méthodes publiques (Getters) :
- GetSexe() : int
- GetPoids() : double
- GetTaille() : double
- GetAge() : int
- GetImg() : double
- GetMessage() : string
- GetImage() : string
