👟 SIO Shoes

Application E-commerce Symfony

⚠️ Projet Symfony

Ce projet nécessite un environnement de développement configuré avec PHP, Composer et Docker pour fonctionner.

📦 Prérequis

  • PHP >= 8.2
  • Composer (gestionnaire de dépendances PHP)
  • Docker et Docker Compose
  • Node.js et npm (optionnel pour les assets)

🚀 Installation en 5 étapes

1

Installer les dépendances PHP

Ouvrez un terminal dans le dossier du projet et exécutez :

$cd Projets/Site_eShoes-main
$composer install
2

Démarrer la base de données

Démarrez Docker Compose pour lancer PostgreSQL :

$docker compose up -d
3

Configurer l'environnement

Créez un fichier .env.local à la racine du projet avec :

$DATABASE_URL="postgresql://app:!ChangeMe!@127.0.0.1:5432/app?serverVersion=16&charset=utf8"
4

Créer la base de données

Exécutez les migrations pour créer les tables :

$php bin/console doctrine:database:create
$php bin/console doctrine:migrations:migrate
5

Lancer l'application

Démarrez le serveur de développement :

$symfony server:start

Ou avec PHP natif :

$php -S localhost:8000 -t public

🌐 Accédez à : http://localhost:8000
