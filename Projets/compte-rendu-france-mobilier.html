Vitrine meubles en HTML statique, et rappel des actualisations du portfolio

1. Projet France Mobilier

Objectif : Reproduire le comportement d’un site vitrine PHP / MySQL (pages d’accueil, liste produits filtrée, magasins, newsletter) sans serveur ni base : tout tourne en HTML, CSS, JavaScript avec des données de démo.
Livrable : Dossier Projets/Francemobilier/ ouvrable en local (index.html).
Auteur : Abdelmalek Elidrissi — BTS SIO SLAM

Fonctionnalités

  • Accueil : carrousel (images locales dans images/carousel/), liste des magasins, formulaire newsletter (démo : alerte, pas d’envoi serveur).
  • Recherche : filtres catégorie et gamme (valeur 0 = toutes), affichage d’un tableau produits (référence, photo, désignation, prix TTC format français).
  • Catalogue : données dans data.js (categories, gammes, produits, magasins).
  • Photos produits : fichiers locaux images/produits/<clé>.jpg (plus d’URL Picsum dans le code).

Technologies (France Mobilier)

HTML5 CSS3 JavaScript Bootstrap 5 Bootstrap Icons

Arborescence indicative

Projets/Francemobilier/ ├── index.html # structure + vues accueil / liste ├── francemobilier.css # compléments à Bootstrap ├── app.js # filtres, rendu tableau, newsletter démo ├── data.js # FM_DATA (JSON de démo) └── images/ ├── carousel/ # salon.jpg, salle-a-manger.jpg, chambre.jpg └── produits/ # canape.jpg, tablebasse.jpg, … (12 fichiers)
